# PROMPTS.md — Antigravity İçin Kullanılacak Hazır Promptlar

Bu dosya, Antigravity ajanına verilecek hazır görev komutlarını içerir.

## 1. Proje İskeleti Oluşturma

```text
Bu workspace için Web Programlama dönem projesi şartlarına uygun bir statik il tanıtım sitesi iskeleti oluştur. Sadece HTML, CSS ve vanilla JavaScript kullan. Bootstrap, Tailwind, jQuery, framework, CDN veya hazır tema kullanma. index.html, assets/css/style.css ve assets/js/main.js dosyalarını oluştur. Zorunlu bölümler: genel tanıtım, tarihçe, coğrafya, kültür, gezilecek yerler, yöresel yemekler, galeri ve iletişim formu.
```

## 2. Responsive Tasarım Geliştirme

```text
Mevcut HTML yapısını bozmadan CSS dosyasını geliştir. Mobile-first responsive tasarım uygula. Kart yapıları mobilde tek sütun, tablet ve masaüstünde çok sütun olsun. Hazır kütüphane kullanma. CSS Grid, Flexbox ve CSS değişkenleri kullan.
```

## 3. Galeri Slider Ekleme

```text
assets/js/main.js içinde vanilla JavaScript kullanarak galeri slider özelliği ekle. Önceki/sonraki butonları çalışsın. Görsel açıklaması değişsin. Kod sade ve açıklanabilir olsun. HTML ve CSS tarafında gerekiyorsa küçük düzenleme yap ama kütüphane ekleme.
```

## 4. Form Validation Ekleme

```text
İletişim/öneri formuna JavaScript ile validation ekle. Ad soyad, e-posta ve mesaj alanları boş geçilemesin. E-posta formatı kontrol edilsin. Mesaj en az 10 karakter olsun. Hatalar ilgili alanın altında gösterilsin. Başarılı durumda kullanıcıya bilgilendirme mesajı gösterilsin. Form gerçek sunucuya gönderilmesin.
```

## 5. Proje Şartları Kontrolü

```text
Projeyi Web Programlama dönem projesi şartlarına göre kontrol et. Sadece HTML/CSS/JS kullanılıp kullanılmadığını, Bootstrap/Tailwind/jQuery/hazır tema olup olmadığını, responsive yapıyı, form validationı, JavaScript dinamik özelliğini ve zorunlu içerik bölümlerini kontrol et. Eksikleri madde madde raporla.
```

## 6. Rapor Notları Oluşturma

```text
Bu projenin raporunda kullanılmak üzere kısa ve resmi bir açıklama hazırla. Kullanılan teknolojileri, site bölümlerini, JavaScript dinamik özelliğini, form validationı ve responsive tasarımı açıkla. Öğrenci projesine uygun sade bir dil kullan.
```
