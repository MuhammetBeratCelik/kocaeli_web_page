## Seçilen İl

`[Kocaeli]`

## Hedef Kullanıcı

- Seçilen ili tanımak isteyen ziyaretçiler
- Turistik/gezi bilgisi arayan kullanıcılar
- Yöresel yemek ve kültür hakkında bilgi almak isteyenler

## Site Amacı

Ziyaretçiye seçilen ilin tarihini, coğrafyasını, kültürünü, gezilecek yerlerini ve yöresel yemeklerini sade ve görsel açıdan güçlü bir şekilde tanıtmak.

## Teknik Kapsam

Bu proje statik web sitesi olarak geliştirilecektir.

Kullanılacaklar:

- HTML
- CSS
- JavaScript

Kullanılmayacaklar:

- Framework
- Kütüphane
- Hazır tema
- Backend
- Veritabanı

## Önerilen Sayfa Yapısı

Tek sayfalık yapı önerilir:

1. Hero / Ana tanıtım
2. Genel bilgiler
3. Coğrafya
4. Kültür
5. Gezilecek yerler
6. Yöresel yemekler
7. Galeri
8. İletişim / Öneri formu
9. Footer

Tek sayfa kullanmak, proje şartlarını daha kolay kontrol etmeyi sağlar. Ancak istenirse çok sayfalı yapı da kurulabilir.
