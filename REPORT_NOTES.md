# REPORT_NOTES.md — Rapor İçin Hazır Notlar

Bu dosya, dönem projesi raporunu doldururken kullanılabilecek kısa notları içerir.

## Proje Konusu

Seçilen il için tanıtım web sitesi hazırlanmıştır. Web sitesinde ilin tarihçesi, genel bilgileri, coğrafi özellikleri, kültürel yapısı, gezilecek yerleri, yöresel yemekleri, görsel galeri bölümü ve iletişim/öneri formu bulunmaktadır.

## Kullanılan Teknolojiler

Projede yalnızca HTML, CSS ve JavaScript kullanılmıştır. Bootstrap, Tailwind, jQuery veya hazır tema kullanılmamıştır.

## Teknik Özellikler

- Responsive tasarım
- Mobil uyumlu navbar
- Kart tabanlı içerik yapısı
- Görsel galeri
- JavaScript ile dinamik galeri slider
- İletişim/öneri formunda validation
- Semantic HTML yapısı

## Form Validation Açıklaması

İletişim/öneri formunda kullanıcıdan ad soyad, e-posta, konu ve mesaj bilgileri alınmaktadır. JavaScript ile boş alan kontrolü, e-posta format kontrolü ve mesaj uzunluğu kontrolü yapılmaktadır. Hatalı girişlerde kullanıcıya uyarı mesajı gösterilmektedir.

## Dinamik Özellik Açıklaması

Projede JavaScript ile galeri slider özelliği geliştirilmiştir. Kullanıcı önceki ve sonraki butonlarıyla şehir görselleri arasında geçiş yapabilmektedir. Ayrıca mobil menü aç/kapat özelliği de JavaScript ile kontrol edilmektedir.
